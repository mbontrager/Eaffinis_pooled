CALL SNPs

 which bcftools
 bcftools --version
 samtools mpileup

The following command works with samtools 0.1.19, for newer version e.g. 1.1.0 see box below.

 samtools mpileup -uD -f genome.fa chr20_5_aln.sorted.bam | bcftools view -bvcg -  > chr20_5_samtools.raw.bcf

Note: NEW COMMAND: create VCF directly (VCF4.2 format)

ALSO in samtools: -D is deprecated => use -t DP

FULL PATH MAY BE REQUIRED DEPENDING ON --version

/mnt/software/samtools-dir/samtools-1.1/bin/samtools mpileup -u -t DP -f genome.fa chr20_5_aln.sorted.bam | bcftools view -v snps,indels,mnps > chr20_5_samtools.raw.vcf

The following command may also work:

/mnt/software/samtools-dir/samtools-1.1/bin/samtools mpileup -u -t DP -f genome.fa chr20_5_aln.sorted.bam | bcftools call -cv - > out-put_2.vcf

CONVERT BCF TO VCF

 bcftools view chr20_5_samtools.raw.bcf | /mnt/software/samtools/bcftools/vcfutils.pl varFilter -D100 > chr20_5_samtools.vcf
 ls -lt *vcf
 head -3 chr20_5_samtools.vcf
 less chr20_5_samtools.vcf
 tail -3 chr20_5_samtools.vcf
 grep -v chr20 chr20_5_samtools.vcf | wc

Quick Editing of VCF file with "vi" editor:

This is no longer required with the newest samtools/bcftools versions creating VCF 4.2 or greater file versions. However, this is a very fancy "in-line" use of the vi editor worth knowing.... sh vi chr20_5_samtools.vcf "+:%s/Number=-1/Number=1/g" "+:wq"

TASK:

cat genome/CP008958.fasta genome/CP008957.fasta > genome/EDL933.fasta

Map reads to reference genome with bwa

bwa index EDL933.fasta
bwa mem -t 4 genome/EDL933.fasta fastq/SRR957857_1.fastq.gz fastq/SRR957857_2.fastq.gz > SRR957857.sam
head -5 SRR957857.sam
```sh

head -4 SRR957857.sam | cat -b sh ls -lh *sam ```

Reorganize SAM into sorted, indexed BAM

samtools view -b -S SRR957857.sam > SRR957857.bam
ls –lh *.bam
samtools sort SRR957857.bam SRR957857_sort
samtools index SRR957857_sort.bam

Identify SNPs

/mnt/software/freebayes/freebayes
/mnt/software/freebayes/freebayes -f genome/EDL933.fasta SRR957857_sort.bam > SRR957857.vcf
